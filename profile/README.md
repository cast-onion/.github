## hey, welcome to cast.onion 👋

cast.onion is a private internet radio network. no public access, no algorithms, no platforms. stations are hand-approved and run by people who actually care about what they're broadcasting.

### 📻 what is cast.onion?

a closed network for internet radio. listeners tune in through the directory. broadcasters apply, get reviewed, and go live with a desktop app. guests can join broadcasts through invite links. everything runs on a session — nothing is open to the public.

### 🖥️ preview

go to https://castonion.xyz and take a look at what is available.

### 🔒 how do i get in?

submit an application through the web interface. if approved, you'll get a station key and access token via email and can start broadcasting immediately.

### 📡 where's the source?

all of our code lives in the monorepo we keep closed source for now.
